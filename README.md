# P.K. ChemFlow (Enterprise ERP & SCM)

## 📌 Project Overview
P.K. ChemFlow is a comprehensive Enterprise Resource Planning (ERP) and Supply Chain Management (SCM) system custom-built for a chemical manufacturing business. 

The system automates core business workflows including dynamic inventory tracking, algorithmic manufacturing optimization, financial ledger generation, and logistics routing. 

## 🚀 Key Features
* **Manufacturing Optimizer Engine:** Implements algorithmic linear programming to calculate the most profitable chemical blending ratios (e.g., Toluene, Benzene) based on fluctuating raw material costs and quality constraints.
* **Automated Finance & Billing:** Generates dynamic, downloadable PDF invoices and tracks real-time GST (Goods and Services Tax) ledgers for IT returns.
* **Smart Procurement & Inventory:** Monitors local stock levels and integrates with external commodity pricing APIs to recommend optimal purchasing windows based on market dips.
* **Logistics & Route Optimization:** Integrates with the Google Maps Distance Matrix API to solve the Vehicle Routing Problem (VRP), calculating the most fuel-efficient delivery sequences for the transportation fleet.

## 🛠️ Tech Stack
* **Backend:** Java 17, Spring Boot 3.x, Spring Data JPA, Hibernate
* **Database:** PostgreSQL (with Relational Entity Modeling)
* **Frontend:** React.js *(Upcoming)*
* **Architecture:** RESTful API, MVC Pattern, Service-Oriented Architecture (SOA)

## 📈 Business Impact
Replaced manual Excel tracking and estimation with a mathematically optimized, automated system, significantly reducing raw material waste, lowering delivery fuel consumption, and eliminating manual billing errors.
